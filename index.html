§ VI · The doctrine

Six commitments. Non-negotiable.

I

Sovereign

Runs on your infrastructure. Your data never leaves. Fork it, host it, run it from disk.

II

MIT-licensed

Every listing. Every schema. No revocation. No rug-pull. Forever forkable.

III

Ed25519-signed

Passports, listings, benchmarks, transactions. Trust is empirical, not marketed.

IV

No telemetry

Zero tracking. Zero analytics. Zero third-party requests. IndexedDB is the boundary.

V

Single-file

One HTML file per application. No build step. No install. Ships as source. Runs from disk. Forkable in one save-as.

VI

Provenance-royalty

Fork the tree, earn from the tree. Improvements propagate. Value tracks lineage.
